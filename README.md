<h1>Food-App</h1>

<h2>A Solo project built by me within 1 day.</h2>
<br>

## Features :

 <p>--> Login</p>
 <p>--> Search recipe</p>
 <p>--> Day of recipe</p>
 <p>--> Recipe all detail</p>





## Tech Stack

- HTML 5
- CSS 3
- Javascript
- JSON Server


Start the server
```bash
  json-server --watch db.json
```
